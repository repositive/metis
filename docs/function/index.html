    public async getSynonyms(payload: String, _postgres: Object): String source 

    import {getSynonyms} from 'metis/src/Synonyms/controllerSynonyms.ts'

    This method is linked to the action iris.synonyms.get. It checks in the database whether a list of synonyms for the given symbol exists. If it does not exist or the last update is older than 30 days, a request is send to the HUGO API at gennames.org to retrieve the required synonyms. The response from the HUGO is stored to the database and returned.

    Params:

    NameTypeAttributeDescription
    payload String

    contains the requested symbol as a string.
    _postgres Object

    Pass postgres element to work with database.

    Return:

    String

    A list of synonyms for the requested symbol.
